# Whereabouts

![logo_title_white](https://user-images.githubusercontent.com/60617744/135721626-a6f23287-a5bd-4f36-8c7d-de589336e9e9.png)

Application written in Flutter framework using Google maps and Firebase. Features push notifications, chats, profiles, and much more!

The main task of the application is receiving location data from the device and then presenting the collected data in real-time on the map. The user is notified about changes in the application by notifications triggered upon meeting certain conditions.
The application also features social functionalities such as profiles, chats, friends list and sharing of discovered places.

# System requirements
<b>Minimal version of android 4.1</b>

- Android SDK minimal version dependency is SDK 21.
- Stable internet connection
- Permisiion to use GPS module
